Cara Pertama.
A. MEMBUKA DENGAN MOZZILA FIREFOX
   1. buka file index.html dengan mozzila firefox
   2. klick tombol (click to download)
   3. secara otomatis akan memberikan notifikasi untuk download file.
   4. bila tidak muncul notifikasi download gunakan cara kedua(B)

Cara Kedua.
B. MEMBUKA DENGAN MENGAKTIFKAN WEB SERVER
   1. aktifkan webserver (xampp, lampp, atau sejenisnya)
   2. pindahkan semua file dan folder (dalam satu folder fuzzyjs) ke folder public server (htdocs)
   3. buka browser masukan alamat (localhost/fuzzyjs/index.html)
   4. klick tombol (click to download)
   5. secara otomatis akan memberikan notifikasi untuk donwload file.
   6. pilih simpan untuk mendownload.
